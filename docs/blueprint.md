# **App Name**: Mekong Scripts Store

## Core Features:

- Dynamic Gaming Storefront: A high-performance product listing page with advanced filtering for frameworks like QBCore, ESX, and standalone scripts, including a search system.
- AI Content Optimizer Tool: An AI tool that uses reasoning to automatically generate compelling gaming-oriented product descriptions and formatted feature lists based on raw script data.
- Cloud Product Repository: A centralized database system to store script metadata, pricing, requirements, and media previews using Firestore.
- Admin Management Command: Secure admin dashboard to perform CRUD operations on products and manage the status of community orders.
- Glassmorphic Product View: Immersive single product pages featuring video embeds, requirement checklists, and sleek price banners with 'Buy Now' triggers.
- Manual Checkout Flow: Order generation system that creates unique transaction IDs and instructions for manual payment via ABA QR or Bank Transfer.

## Style Guidelines:

- Primary color: Electric Cyan (#00E5FF), a saturated neon blue that provides high contrast for interactive elements against the dark backdrop.
- Background color: Midnight Obsidian (#0B0E14), a deep, desaturated blue-black that emphasizes the 'cyber' aesthetic.
- Accent color: Cyber Violet (#D100FF), a vivid analogous hue used for secondary actions and decorative glows to enhance the gaming feel.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-heavy, futuristic look. Body font: 'Inter' (sans-serif) for high legibility across product details and documentation.
- Lucide-style line icons with thin strokes, modified with an external glow filter to match the neon theme.
- Glassmorphism based cards with 10% opacity fills, 1px cyan borders, and 'backdrop-filter: blur(12px)' to create depth on dark backgrounds.
- Framer Motion transitions including smooth scale-up effects on hover and 'pulse' animations on neon buttons to signify interactivity.
